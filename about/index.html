关于博客

喜欢写 Blog 的人，会经历三个阶段：

  • 第一阶段，刚接触 Blog，觉得很新鲜，试着选择一个免费空间来写。

  • 第二阶段，发现免费空间限制太多，就自己购买域名和空间，搭建独立博客。

  • 第三阶段，觉得独立博客的管理太麻烦，最好在保留控制权的前提下，让别人来管，自己只负责写文章。

我曾接触过博客园，CSDN，掘金等这类免费空间，也曾自己购置过阿里云的空间域名，搭建 WordPress，甚至基于 xiuno 搭建过论坛，但这第一阶段的免费空间，限制太多；第二阶段的空间域名，花费太大，对于当时还是学生的我，同样不太适合，最后我迈入了第三阶段。

了解过 jekyll 和 hexo 后，我毅然选择了性能更好的后者，但对于我这个选择困难户来讲，我在主题上纠结了好久，再三抉择之后先选择了最火的 Next，但是折腾了一两天后，突然觉得菜单 page 没有自己链接到菜单项，一切重新配置似乎又回到了第二阶段的繁杂期，这可不是我想要的，毕竟我可是个极简主义者（主要前端太烂），之后便兜兜转转用了一段时间的 cactus，结果某一天突然就崩了，现在想想还是之前 Git HTTP 代理的历史遗留问题没有解决，导致无法提交，因缘际会之下，最后换成了当前的主题，如梦幻般的星空，群星闪烁……

未来的话，应该会一直完善这个主题，毕竟我很喜欢哒！

感兴趣的朋友，主题是在 hexo-Fan 的基础上进行修改的，欢迎一起完善 ^_^

    Chrome 快捷键整理

    打开主页 Alt + Home

    打开新窗口 Ctrl + n
    无痕模式下打开新窗口 Ctrl + Shift + n
    用户 Ctrl+shift+m
    打开新的标签页，并跳转到该标签页 Ctrl + t

    跳转到下一个打开的标签页 Ctrl + Tab 或 Ctrl + PgDn
    跳转到上一个打开的标签页 Ctrl + Shift + Tab 或 Ctrl + PgUp
    跳转到特定标签页 Ctrl + 1 到 Ctrl + 8

    关闭当前标签页 Ctrl + w 或 Ctrl + F4
    关闭当前窗口 Ctrl + Shift + w 或 Alt + F4（关机）

    打开 Chrome 菜单 Alt + f 或 Alt + e

    打开“历史记录”页 Ctrl + h
    打开“下载内容”页 Ctrl + j

    打开查找栏搜索当前网页 Ctrl + f 或 F3
    打开“开发者工具” Ctrl + Shift + j/i 或 F12
    显示当前网页的 HTML 源代码（不可修改） Ctrl + u

    打开“清除浏览数据”选项 Ctrl + Shift + Delete

    跳转到地址栏 Ctrl + l Alt + d 或 F6

    将当前网页保存为书签 Ctrl + d
    将所有打开的标签页以书签的形式保存在新文件夹中 Ctrl + Shift + d
    显示或隐藏书签栏 Ctrl + Shift + b
    打开书签管理器 ctrl+shift+o

    停止加载网页 Esc
    放大/缩小网页上的所有内容 Ctrl 和 +/-
    将网页上的所有内容恢复到默认大小 Ctrl + 0

    window + 上下左右键
    在任务栏中快速切换程序 Win + 1
    创建一个新的文件夹 Ctrl + Shift + N
    显示桌面 win+d/m
    显示设置 win+i
    文件资源管理器 win+e
    左侧工具栏 win+x
    右侧设置 Win+a

    Vimium 快捷键

    所有快捷键 ?
    向下/上/左/右移动 j/k/h/l
    后退/前进网页 H/L
    向下/上跳动 d/u-e
    回到顶/尾部 gg/G
    窗口打开模式 本窗口/新窗口 f/F
    （如果输入的是小写的英文则会在当前标签打开，想要打开新的标签页要输入大写的英文，即按住shift再输入）
    打开新页面并切换 S
    查找历史记录+书签 o/O
    查找书签 b/B
    关闭/恢复标签 x/X-z
    切换到刚才的标签 ^-q
    移动当前标签到左/右侧边 <<-</>>->
    将标签页移动到新窗口 W-ww
    选择左/右标签 J/K
    搜索剪贴板关键字 在当前/新窗口 p/P
    创建/查看标签页 t/T（数字nt）
    将焦点聚集在第一个输入框 gi (2gi就是第二个输入框)
    刷新 r
    新标签中打开多个链接 即：alt+f
    开/关静音 即：alt+m
    固定标签栏 即 alt+p

    ]], [[ 下一个/上一个
    yt 复制当前页面
    yy 将当前的网址复制到剪贴板
    yf 显示链接字母，并将网址拷贝到剪贴板

    ma 当页标记，只能在当前tab页面跳转，m + 一个小写字母
    mA 全局标记，可以再切换到其他tab的跳转过来，m + 一个大写字母
    a 跳转到当页标记A 跳转到全局标记

    编辑当前的网址 ge/gE
    跳转到当前url上一级/最高级 gu/gU
    （需按住shift切换后面的大写）
    g0 切换到第一个tab
    g$-g9 切换到最后一个tab
    gs 显示网页源代码

    i 进入插入模式，所有按键的命令都无效，直至ESC键退出
    / 进入查找模式，输入关键字查找，ESC退出
    n/N 切换到下/上一个匹配

    Visual Studio 快捷键

    ctrl+shift+a 新建项
    alt+shift+a 现有项

    ctrl+b 生成
    F10 逐过程
    F11 逐语句
    ctrl+alt+d 反汇编
    ctrl+delete 删除整行

    【Google hack之精简部分】
    1.查看基本情况
    info:xx.com 返回一些基本信息
    site:xx.com 返回所有与该有关的url
    link:xx.com 返回所有与该站做了链接的站
    site:xx.com filetype:txt 查找txt文件

    2.查找后台
    site:xx.com intext:管理
    site:xx.com inurl:login
    site:xx.com intitile:后台

    3.查看服务器使用的程序
    site:xx.com filetype:asp
    site:xx.com filetype:php
    site:xx.com filetype:jsp
    site:xx.com filetype:aspx

    4.查看上传漏洞
    site:xx.com inurl:file
    site:xx.com inurl:load

    5.查找注射点
    site:xx.com filetype:asp

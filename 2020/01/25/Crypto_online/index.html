Quoted-printable编码

我们收邮件，查看信件原始信息，经常会看到这种类型的编码！（=E6=95=8F=E6=8D=B7=E7=9A）
function quoted_printable_encode($string) {
return preg_replace(‘/[^\r\n]{73}[^=\r\n]{2}/‘, “$0=\r\n”, str_replace(“%”,”=”,
rawurlencode($string)));
}
